# Les talk

## What is  Les talk ?

A front-end only application which allow users to speak together on a shared computer.

## Why Choose Les talk?

- messages betwen you and trusted Laptop co-owner are kept safe.
- easy to use and access.
- Ad free.


# Documentation

## Software Requirement Specification

### Overview

Les Talk is a real-time chat application that allows users to communicate via private messages and group chats. Users can sign up, log in, update their profile, and chat with other online/offline users.

### components and functional requirement

**1. User Authentication**
  * Users are able to sign up.
  * Users are able to log in. 
  * Users are able to log out. 

**2. Real-Time Chat Features**
 * Displays online users.
 * Enables users to send and receive messages in real-time. 
 * Supports both group and private chat.
